Privacy Policy

ManageBac GPA Calculator

This Privacy Policy explains what information the extension can access, how it is used, and how user data is handled when the extension runs on supported ManageBac pages.

Effective Date: April 19, 2026 Applies to Chrome Extension Use Local-Only Processing

Overview 概述

ManageBac GPA Calculator is a Chrome extension that helps users scan visible ManageBac gradebook tags, calculate weighted GPA or percentage results, display inline summary badges, and generate student trend charts.

ManageBac GPA Calculator 是一款 Chrome 扩展，用于扫描 ManageBac 成绩簿页面中可见的标签， 计算加权 GPA 或百分比结果，显示行内成绩气泡，并生成学生趋势图。

Important: All gradebook processing happens locally in the user's browser. The extension does not upload gradebook content to any external server.
重要说明：所有成绩数据处理均在用户浏览器本地完成，扩展不会将成绩簿内容上传到任何外部服务器。

Data the Extension Can Access 扩展可能访问的数据

When a user opens the extension on a supported ManageBac page, the extension may read visible page content that is necessary to provide its features, including:

当用户在受支持的 ManageBac 页面上使用本扩展时，扩展可能会读取提供功能所必需的、当前页面可见内容，包括：

  • Student names / 学生姓名
  • Assignment titles / 作业标题
  • Assignment dates / 作业日期
  • Visible tags and categories / 可见标签与分类
  • Scores, percentages, and grade symbols / 分数、百分比与字母等级
  • Course or subject information shown in the gradebook / 成绩簿中显示的课程信息
  • User-selected rules, tag groups, and weight settings / 用户选择的规则、标签分组与权重设置
  • Temporary trend chart data used to open the trend view / 用于打开趋势图页面的临时趋势数据

How the Data Is Used 数据如何使用

The extension uses accessed data only to provide the user-requested features of the extension.

扩展访问这些数据的唯一目的，是实现用户主动使用的扩展功能。

  • Scan ManageBac tags from the visible gradebook / 从当前可见成绩簿扫描标签
  • Calculate weighted GPA or percentage results / 计算加权 GPA 或百分比结果
  • Display inline badges and trend buttons / 显示成绩气泡与趋势按钮
  • Generate trend charts and PDF-ready reports / 生成趋势图与可导出的 PDF 报告
  • Save local rule settings for later reuse / 保存本地规则设置以供后续复用

Storage and Retention 存储与保留

The extension uses Chrome local storage only to save extension settings and temporary data required to open the trend view.

本扩展仅使用 Chrome 本地存储来保存扩展设置，以及打开趋势图页面所需的临时数据。

  • Selected tag groups / 已选标签分组
  • Rule weights / 规则权重
  • Score conversion mode / 分数换算模式
  • Temporary trend chart payload / 临时趋势图数据

Stored data remains in the user's browser environment and can be removed by clearing extension data or uninstalling the extension.

这些数据保留在用户浏览器本地环境中，用户可以通过清除扩展数据或卸载扩展来移除它们。

Data Sharing 数据共享

The extension does not sell, transfer, disclose, or share personal data or gradebook data with third parties.

本扩展不会向第三方出售、转移、披露或共享任何个人数据或成绩簿数据。

  • No remote analytics / 不使用远程分析服务
  • No cloud sync / 不使用云同步
  • No advertising network integration / 不接入广告网络
  • No external server upload of ManageBac gradebook content / 不向外部服务器上传 ManageBac 成绩内容

Permissions 权限说明

  • activeTab: used to access the active ManageBac tab after user interaction. / 在用户触发操作后访问当前激活的 ManageBac 标签页。
  • scripting: used to inject the extension's bundled content script and CSS into supported ManageBac pages. / 用于向受支持的 ManageBac 页面注入扩展自带的脚本和样式。
  • storage: used to save local extension settings and temporary trend data. / 用于保存本地扩展设置和临时趋势数据。
  • Host access: limited to managebac.com and managebac.cn pages only. / 主机访问范围仅限于 managebac.com 和 managebac.cn 页面。

Compliance Statement 合规声明

The use of information received from Chrome APIs and ManageBac pages will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.

通过 Chrome API 和 ManageBac 页面获得的信息的使用，将遵守 Chrome Web Store 用户数据政策， 包括 Limited Use 相关要求。

Contact 联系方式

If you have questions about this Privacy Policy, please contact the publisher using the support email shown on the Chrome Web Store listing for ManageBac GPA Calculator.

如果您对本隐私政策有疑问，请通过 Chrome Web Store 中 ManageBac GPA Calculator 商店页面显示的支持邮箱联系发布者。
