🔒 数据处理方式 Data Processing Method

纯本地处理 Local-Only Processing

本扩展程序仅在用户浏览器本地处理数据。所有学生成绩数据、姓名信息仅在用户设备上进行可视化分析和 CSV 导出，不会传输至任何远程服务器、云存储或第三方服务。

This extension processes all data locally within the user's browser only. All student grade data and personal information are visualized and exported as CSV on the user's device only, without transmission to any remote servers, cloud storage, or third-party services.

📊 数据收集范围 Data Collection Scope

当您主动点击扩展图标时，本扩展会读取当前打开的 ManageBac 成绩簿页面中的以下信息：

  • 学生姓名 Student names
  • 学科/课程信息 Subject/course information
  • 作业/考试成绩 Assignment/exam scores
  • 百分比等级 Percentage grades

注意：仅读取您已授权访问的、已打开的 ManageBac 页面内容。

💾 存储说明 Storage

使用 Chrome Storage API 本地保存：

  • 用户图表偏好设置（如分组方式、学期选择）
  • 临时提取的成绩数据（仅在当前浏览器会话期间）

可随时通过"清除浏览器数据"或卸载扩展删除所有存储内容。

🔐 权限使用说明 Permission Usage

  • activeTab: 识别当前激活的 ManageBac 成绩页
  • host权限 (*.managebac.com): 仅访问 ManageBac 域名以读取成绩数据
  • scripting: 在 ManageBac 页面执行数据提取脚本
  • storage: 本地保存用户设置和临时数据
